### System requirements
- node v6.11.3
- react 15.6.2
- react-router-dom 4.2.2 (React Router v4)

### Start both API and UI servers
$ npm start

### Run unit tests
$ npm test

### Run unit tests with coverage
$ npm test -- --coverage
